# e1-prompts-and-workflows

This repository is for managing prompt engineering deliverables, as well as notes and configuration files related to personal workflows.

## Overview
This project records and shares custom prompts (system prompts and module definitions) to better leverage advanced language models and optimize AI agent behavior.

## Contents

- **[`prompts/any_ai_deep_research_prompt.md`](./prompts/any_ai_deep_research_prompt.md)** / **[`.xml`](./prompts/any_ai_deep_research_prompt.xml)**
  A custom-developed system prompt dedicated to **Deep Research** using any LLM, including **DeepSeek**. You can run it by pasting it into the chat input of any browser-based AI and sending it after appending your topic at the end.
  It is integrated and optimized based on the system prompts of the following powerful autonomous research AIs and agents:
  - [OpenManus](https://github.com/mannaandpoem/OpenManus)
  - [GPT-Researcher](https://github.com/assafelovic/gpt-researcher)
  - [STORM](https://github.com/stanford-oval/storm)
  - [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT)
  * Two formats are provided—Markdown and XML—so you can choose based on execution environment and parsing requirements.

- **[`docs/scrapbox/`](./docs/scrapbox/README.md)**
  Synchronized notes mirroring public pages from the Scrapbox project ellimissinina on GitHub. See [`docs/scrapbox/README.md`](./docs/scrapbox/README.md) for details.

## Purpose
- Improving the accuracy of autonomous Deep Research
- Version control of prompts for complex reasoning tasks
- Experimenting with and recording new workflows
